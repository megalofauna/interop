Interop builds on what the platform already provides — native elements, CSS cascade, and browser semantics — rather than replacing them. The result is components that are correct without effort.

HTML-first

Attribute selectors attach to native elements. <button interop-button> keeps keyboard behavior, ARIA role, form participation, and focus management for free — none of it needs to be recreated.

Angular-forward

Signals are the reactive primitive — no subjects, no manual change detection, no markForCheck(). Every component is standalone, OnPush, and uses the new control flow syntax. Not aspirationally. Actually.

Ships only what you use

Import InteropButton and you pay for InteropButton. Nothing else. Every component, icon, and adapter is tree-shakeable by design. Heavy dependencies like syntax-highlighting grammars are lazy-loaded on demand.

A token system that compounds

Override --itx-control-accent at any ancestor and every form control in that subtree updates — checkbox, radio, and toggle simultaneously. No selector specificity battles. The cascade does the work.
