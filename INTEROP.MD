# Interop

## What is Interop
Interop is a TypeScript-based component library for the Angular Web framework. 

## Priorities of Interop
 - Refreshing commonalities across APIs
 - Unparalleled developer experience
 - Easy to learn, just as easy to predict
  
 - Optimized for agentic workflows


 ## Common API
 ### Common properties across the component library



 `Collection` 
 Any iterable collection (?) - what does this include?
 An array of arbitrary same-typed items passed to a component for rendering in the UI.
